    Una computadora u ordenador, es un equipo electrónico compuesto de diferentes partes que funcionan de manera lógica para permitir a los usuarios la posibilidad de automatizar todo tipo de tareas.

    Componenetes de una computadora

    Para que todo esto sea posible, una PC debe contar con soporte de hardware, el cual no es más que toda la estructura física compuesta de piezas y periféricos que necesita el equipo para operar de manera idónea.

    Otorgan un equilibrio en la computadora para hacer que funcione correctamente. Estas se dividen principalmente en tres partes el hardware, software y firmware.
